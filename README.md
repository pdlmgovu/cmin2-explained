## 二、CMIN2 是什么线路？三句话说清楚

**CMIN2**，全称是 **China Mobile International Next-generation Network**，ASN 号是 **AS58807**，是中国移动自己建的高端国际骨干网。

你可以把它理解成：移动版的 CN2 GIA。

以前移动出海只有普通的 CMI（AS58453），带宽不算多、高峰容易拥挤。CMIN2 是移动专门为了解决这个问题新建的精品通道，跟国内的移动骨干网（AS9808）直接对接，绕开公共交换节点，延迟低、丢包少。

> **一句话总结**：CMIN2 = 移动用户的高端出海专线，等同于电信的 CN2 GIA 地位。

区分普通 CMI 和 CMIN2 的方法很简单——看 ASN：
- AS58453 = 普通 CMI（不要）
- **AS58807 = 精品 CMIN2（这个才是你要的）**

## 八、几个新手常犯的错误

**误区 1：看到"CN2"就以为是 CN2 GIA**

CN2 分 GT 和 GIA 两种，CN2 GT（AS4809 Global Transit）高峰期早就堵得和普通线路差不多了。只有 CN2 GIA（Global Internet Access）才是真正的精品线路。搬瓦工的 Basic 套餐走的是普通国际线路，不要搞混。

**误区 2：以为 CMIN2 是 CMI**

两个不同的东西：AS58453 是普通 CMI，AS58807 才是 CMIN2。普通 CMI 高峰期容易堵，CMIN2 才是移动自己的精品骨干，区别很大。

**误区 3：买了套餐却用不到 CMIN2**

CMIN2 目前只在 CN2 GIA-E 套餐下的 DC6、DC9、EUNL_1 等机房可用。买了 Basic 套餐或者其他普通套餐，是享受不到 CMIN2 的。
